# Hotel Sachida Palace Booking System

A complete hotel booking website for Hotel Sachida Palace built with MERN stack (MongoDB, Express, React, Node.js).

## Features

- 🏨 Room listings with details (Double Bed, Triple Bed, Family Suite)
- 📅 Real-time availability checking
- 💳 Payment integration (UPI, Cards, NetBanking via Razorpay)
- 📧 Email confirmations to guest and owner
- 💬 WhatsApp notifications to guest and owner
- 👑 Admin dashboard for managing bookings
- 📱 Fully responsive design

## Room Details

| Room Type | Price | Capacity |
|-----------|-------|----------|
| Double Bed | ₹2,000 | 2 Adults + 1 Child |
| Triple Bed | ₹2,500 | 3 Adults + 1 Child |
| Six Bed Suite | ₹4,500 | 6 Adults + 3 Children |

All rooms include attached toilet and geyser.

## Prerequisites

- Node.js (v14+)
- MongoDB
- Razorpay Account (for payments)
- Gmail Account (for email)
- Twilio Account (for WhatsApp)
